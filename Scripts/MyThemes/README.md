## 使用方法：

#### 通过 curl

```shell
sh -c "$(curl -fsSL https://raw.githubusercontent.com/Omico/OhMyScripts/master/Scripts/MyThemes/MyThemes.sh)"
```

#### 通过 wget

```shell
sh -c "$(wget https://raw.githubusercontent.com/Omico/OhMyScripts/master/Scripts/MyThemes/MyThemes.sh -O -)"
```
